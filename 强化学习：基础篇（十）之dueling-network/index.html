Dueling Network

Dueling Network[1]方法是对DQN网络结构的一种改进，同样可以大幅提升DQN的表现。当然，Dueling Network不局限于在DQN中使用，还可以用于其他方法。

优势函数（Advantage Function）

在讲Dueling Network之前，需要引入Advantage的概念。

首先回顾Value Function的定义：

  • 首先有折扣回报
  • 其次可以定义状态价值函数
  • 最后得到状态价值函数

有了Value Function之后就有Optimal Value Function：

  • 最优动作价值函数
  • 最优状态价值函数
  • 接着可以定义最优优势函数(Optimal Advantage Function)，这里的实际上称为Baseline，而最优优势函数相当于动作相对于Baseline的优势，于是动作越好，优势则越大。
